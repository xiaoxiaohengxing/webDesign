Installation and uninstallation
---------------------------------------------------------------------------

Important! If you've got 123 Flash Menu not from our web page, but from the
other source (magazine CD or some software library), please visit our home
page - you'll probably find the later version;

Installation
Download the latest version of 123 Flash Menu at http://www.dreamingsoft.com.
Extract the downloaded zip file into a temporary directory, then execute the
setup.exe.

Note: 123 Flash Menu will first check if Macromedia Flash Player was installed
on your computer. In case you have not installed it, please follow the on
screen instructions to install the latest version of Macromedia Flash Player.

Uninstallation
You can remove 123 Flash Menu from your system as follows:
1. Open the Windows Control Panel.
2. Double click on Add/Remove Programs.
3. Find 123 Flash Menu in the list, and double click on it.
4. Follow the on screen instructions to remove it.
